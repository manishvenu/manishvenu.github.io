CrocoDash is a Python software package developed under the [CROCODILE-CESM](https://github.com/CROCODILE-CESM) project at the National Center for Atmospheric Research (NCAR). It provides a streamlined workflow for configuring regional MOM6 ocean model simulations within CESM, enabling researchers to set up high-resolution regional ocean configurations without manually managing the full complexity of the model infrastructure.

**Key features:**
- Automated setup of regional MOM6 configurations within CESM
- Python-based tooling with conda/mamba environment management
- Comprehensive documentation and test suite
- Apache 2.0 licensed
