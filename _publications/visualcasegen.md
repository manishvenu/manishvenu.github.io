visualCaseGen provides an interactive, constraint-solver-assisted interface for configuring Community Earth System Model (CESM) experiments. By encoding model configuration rules as SMT constraints, it prevents invalid experiment setups and guides users to valid configurations — making CESM more accessible to researchers without deep model internals expertise.
